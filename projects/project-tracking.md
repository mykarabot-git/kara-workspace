## How It Works

### Projects Dashboard (`projects/INDEX.md`)
- Quick status view of all projects
- Links to individual project files
- Recent activity feed

### Project Files (`projects/<name>.md`)
- Overview and context
- Current state checklist
- Task breakdown by phase
- Running log of progress

### Work Log (`WORK-LOG.md`)
- Chronological record of completed work
- What we did, when, and any notes
- Proof of accomplishments
